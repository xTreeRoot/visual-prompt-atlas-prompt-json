## 稳定 ID

- 每条场景记录都有稳定 `id`，格式为 `scene_0000`。
- 首轮 id 按当前数组位置回填；后续新增场景应通过 `python3 scripts/visual_prompt_atlas.py ingest scenes <file> --write` 分配新 id。
- 数组可以重排，但旧 id 不应重排或复用。使用 `compose --scene-id scene_0000` 可锁定指定场景。

## 服装适配字段

- `compatible_outfit_occasions`：该场景可正向匹配的服装 `occasion` 或 `original_occasion`。
- `forbidden_outfit_keywords`：该场景中明显违和的服装关键词，例如公共空间禁用泳装类关键词。
- 这些字段属于场景库数据，不应再作为 Python 硬编码词表维护。

## 使用建议

1. 如果目标是做稳定的日常对话背景，优先从 **卧室 / 客厅 / 人行道 / 城市夜景平台** 里选。
2. 如果目标是做陪伴感或私聊感，优先使用 **居家私密** 分类，并围绕 `安静`、`私密`、`亲密感` 组合。
3. 如果目标是做都市过渡镜头，优先使用 **城市街头** 分类，并围绕 `日常感`、`流动场景`、`真实城市环境` 组合。
4. 如果目标是做松弛、等待、垂钓感内容，优先使用 **户外自然** 分类，并围绕 `宁静`、`等待`、`开阔空间` 组合。
5. 需要避免穿帮时，可先检查该场景的推荐动作是否与角色姿态一致，再对照高频禁止动作筛掉不合理行为。
